Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The head element contains metadata used in search engines and optimization.
  • The body element represents the visible content shown to the user.
  • The footer is a part of the body and is just another way to display content.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • There are 3 types of CSS inline, internal and external. Prefer to use external and link it.
  • Can define CSS criteria by class and assign those classes to whichever selected elements you want it to.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git and git hub are different, github is a repo
  • touch is used to create files or folders
  • git checkout branchname filename can be used to duplicate files in the current branch
